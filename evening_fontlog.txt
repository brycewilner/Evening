Fontlog for Evening

This file provides detailed information on the Evening font software. This information should be distributed along with the Evening font and any derivative works.

Basic Font Information

Evening is a Unicode typeface family that supports all languages that use the Latin script and its variants, and could be expanded to support other scripts.
